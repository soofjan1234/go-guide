这个面试官的风格不是问概念定义，而是给一个具体场景，然后连续追问“你怎么定位、你凭什么这么判断、失败了怎么办、底层到底发生什么”。如果还是这个人来面，比较可能继续问下面这些题。

1. 线上事故排查

    1. 现在有一个接口 P99 突然从 100ms 涨到 2s，但是错误率没涨，你怎么定位？
    2. 线上只有少量用户报错，日志里没有明显 error，你怎么把这个用户的一次请求链路串起来？
    3. 如果没有链路追踪，你会如何用 request_id / trace_id / 日志字段临时补救？
    4. 数据库 CPU 不高，但是接口很慢，可能是什么原因？
    5. 你发现最近发版后问题变多，但不能马上回滚，你会怎么止血？
    6. 如果是第三方接口偶发超时，你会怎么做超时、重试、熔断和降级？

2. Go 并发 / Goroutine 生命周期

    1. 如何优雅关闭一组 worker Goroutine？任务队列、close channel、WaitGroup、context 怎么配合？
    2. Goroutine 泄漏怎么发现？pprof 里你会看什么？

3. Context

    1. `context.WithValue` 适合放什么，不适合放什么？
    2. 为什么不建议把业务大对象、可变对象、连接对象塞进 context？

4. Channel

    1. 用 channel 做任务队列时，消费者处理不过来会怎样？如何做背压、丢弃、限流？

5. Slice / Map / 并发安全

    1. 函数 A 把 slice 传给函数 B，B 要删除元素，你如何避免影响 A？

6. GMP / netpoll / syscall

    1. P 被 syscall 里的 M 带走后，其他 G 怎么继续跑？
    4. netpoller 发现 FD 可读后，怎么把对应 Goroutine 放回可运行队列？

7. MySQL

    1. explain 里 `type` 字段有哪些常见值？`ALL`、`index`、`range`、`ref`、`const` 分别代表什么？
    2. `Using index` 和 `Using where` 分别是什么意思？
    3. 什么是覆盖索引？为什么能减少回表？
    4. 联合索引的最左前缀原则是什么？遇到范围查询后为什么后面的列可能用不上？
    5. 当前读和快照读有什么区别？`select ... for update` 走哪一种？
    6. RR 隔离级别下，MVCC 能解决幻读吗？什么时候还需要 next-key lock？
    7. undo log、redo log、binlog 分别解决什么问题？

8. Redis

    1. RDB、AOF、混合持久化分别适合什么场景？
    2. AOF rewrite 是怎么做的？rewrite 期间新写入的数据怎么办？
    3. Redis key 过期后是不是立刻删除？如果大量 key 同时过期会怎样？
    4. 缓存穿透、击穿、雪崩分别怎么解决？
    5. Redis 单线程为什么还能快？哪些操作会阻塞 Redis？
    6. 热 key 怎么发现和处理？

9. TCP / 网络

    1. 为什么主动关闭方进入 TIME_WAIT？
    2. CLOSE_WAIT 很多说明什么问题？
    3. 大量短连接会带来什么问题？如何用连接池或 keep-alive 优化？
    4. TCP 粘包是什么？应用层如何解决？
    5. HTTP 超时应该设置哪些：连接超时、读超时、写超时、整体超时？
