## IP 地址和子网掩码 +1

![IP.组成](pic/IP.组成.png)

IP是由网络号和主机号组成的

![IP.子网掩码](pic/IP.子网掩码.png)

子网掩码与IP地址相与得到网络号

## IP 和 MAC 的区别？ +1

![IP与MAC.区别](pic/IP与MAC.区别.png)

IP 端到端寻址，MAC 链路层下一跳寻址；跨网段时目标 IP 不变，MAC 会随每一跳变化

## 交换机和路由器的区别？ +1

![交换机与路由器.区别](pic/交换机与路由器.区别.png)

交换机是数据链路层的，针对MAC地址；

路由器是网络层的，针对IP地址
