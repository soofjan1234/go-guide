## Docker

Docker 项目主要封装 Docker Engine API，提供镜像、容器、网络和基础状态管理能力。比如查询本地镜像、删除镜像、创建容器、启停容器、查询容器状态、端口映射、挂载目录、环境变量配置等。

拉镜像是长耗时任务，所以接口先创建任务并返回 taskId，后端异步执行 docker pull，同时读取 Docker daemon 返回的 JSON stream。前端通过 SSE 订阅这个 taskId，后端把每一段进度事件转成 SSE event 推给前端。

SSE 比轮询实时性好、实现简单；如果要双向控制才考虑 WebSocket
